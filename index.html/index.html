    AI導入の3フェーズ

    「わかる」で終わらない、 その先の「ブレイクスルー」へ

    企業のAI導入には、誰もが通る3つのフェーズがあります。

    第一フェーズ：AIへの抵抗をなくし、まずは気軽に触れてもらうこと。この教材の目的は、まずここにあります。

    第二フェーズ：業務で積極的にAIを使うようになること。多くの企業がこの「活用の壁」にぶつかります。私たちは、この壁を乗り越えるための運用と仕組み作りを、徹底的に支援します。

    そして、この壁を超えた時、AIを完全に使いこなし、ビジネスにブレイクスルーが訪れます。

    私たちが選ばれる理由

    人材育成と経営視点を掛け合わせた、唯一無二の価値を提供します。

    理由１

    最高品質の研修教材

    東証一部上場企業『㈱ライトアップ』と、チャンネル登録者数17万人超のAI専門家『いけとも先生』が共同開発。いつでもどこでも、最新かつ最高品質のAI知識を学べます。

    理由２

    破格の助成金申請支援

    通常150万円が相場の助成金申請支援を、業界の常識を覆す価格で提供。条件次第では着手金・計画申請代行が無料で、成功報酬は入金額のわずか5%のみ。

    理由３

    弊社からのスペシャル特典

    研修導入企業様には、全国180万社中12000社が活用した、実質的な費用負担をさらに軽減する国の制度を最大限活用するノウハウを直接コンサル。貴社の未来への投資を、全力で後押しします。

    理由４

    現場密着の伴走支援

    AI講師、導入支援コンサルタントとして、貴社に深く密着。理念に掲げる「社会循環の改善」と「地方創生」の実現に向け、誠心誠意、貴社だけの最適解を導き出します。

    助成金を活用した、賢い未来への投資

    今期1000万円の利益が出た場合…

    資金繰り比較図

    今期はリスキリング助成金で【偶然にも】上記のような有利な資金繰り（節税効果）が適用されます。来期も同様のことをしても400万円をずらしているだけとなり節税効果はありません。浮いた資金を、来期の事業展開を加速させるための投資に充当しませんか？

    受講までのステップ

    1. 受講者の選定: 雇用保険加入者が対象です。
    2. 書類の準備: 法律上の労務書類と研修申込書をご提出いただきます。
    3. 計画申請: 国へ事前の計画申請を行います。
    4. 研修受講: 計画申請の約1ヶ月後から受講開始です。

    受給の主な条件

    • 雇用保険加入者を1名以上雇用していること
    • 従業員の保険加入義務に違反していないこと
    • 半年以内に会社都合解雇をしていないこと
    • 残業代未払いなどの労務違反がないこと
